COMO INSTALAR (macOS)
=====================

1. Abra o Terminal:
   - Pressione Cmd + Espaço
   - Digite: Terminal
   - Pressione Enter

2. Cole o comando abaixo no Terminal e pressione Enter:

   bash <(curl -sL https://raw.githubusercontent.com/filipegajo89/anki-tec/main/install_mac.sh)

3. Siga as instruções na tela (~15 minutos)


COMO INSTALAR (Windows)
========================

1. Abra o PowerShell como Administrador:
   - Pressione a tecla Windows
   - Digite: PowerShell
   - Clique com botao DIREITO em "Windows PowerShell"
   - Selecione "Executar como administrador"
   - Clique Sim

2. Cole o comando abaixo e pressione Enter:

   Set-ExecutionPolicy Bypass -Scope Process -Force; irm https://raw.githubusercontent.com/filipegajo89/anki-tec/main/install_windows.ps1 | iex

3. Siga as instrucoes na tela (~15 minutos)


O QUE O SCRIPT FAZ AUTOMATICAMENTE
====================================

  * Instala Google Chrome (navegador)
  * Instala Anki (flashcards)
  * Instala Obsidian (notas de estudo)
  * Instala Node.js (necessario para MCPs)
  * Instala uv + notebooklm-mcp-cli (para podcasts)
  * Cria pastas necessarias
  * Configura Claude Desktop


O QUE VOCE PRECISA FAZER MANUALMENTE
======================================

O script avisa e abre os links quando necessario:

  1. Instalar Tampermonkey no Chrome (extensao)
  2. Instalar o script TEC no Tampermonkey
  3. Instalar AnkiConnect no Anki (codigo: 2055492159)
  4. Instalar Local REST API no Obsidian (plugin)
  5. Gerar chave de IA:
     - Opcao A: Gemini (gratuita) em aistudio.google.com/apikey
     - Opcao B: OpenRouter (tem modelo gratis) em openrouter.ai/keys
  6. Configurar o script (botao engrenagem no TEC)


COMO USAR DEPOIS DE INSTALADO
===============================

Para gerar flashcards:
  1. Abra o Anki e o Obsidian
  2. No TEC Concursos, resolva um caderno
  3. Na questao que errou: pressione Shift + Enter
  4. Ou clique em "Erros" para processar todas de uma vez

Para gerar podcasts:
  1. Abra o Claude Desktop
  2. Diga: "Leia minhas notas de Direito Constitucional
     e gere um podcast no NotebookLM"


PROBLEMAS COMUNS
=================

  * "Shift+Enter nao faz nada"
    → Anki e Obsidian precisam estar abertos

  * "Erro de conexao com Anki"
    → Verifique se AnkiConnect esta instalado

  * "O script nao aparece no TEC"
    → Verifique se o Tampermonkey esta ativo no Chrome
  * "Gemini erro 503"
    -> Servidores sobrecarregados. Troque para OpenRouter
       nas configuracoes (engrenagem) ou aguarde
  * "nlm login falhou"
    → Abra o Terminal e rode: nlm login
